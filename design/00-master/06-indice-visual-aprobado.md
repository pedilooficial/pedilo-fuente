# Índice visual aprobado — Usuario público Pédilo

Este índice define las referencias visuales aprobadas para reconstruir el usuario público.

## Regla general

Las imágenes son referencias de diseño.
No son recursos runtime para pegar como pantallas.
No usar como `Image(... fillMaxHeight ...)` ni como screenshot navegable.

## Splash

```text
design/Splash/
  1-splash.png      → pantalla vertical final del splash / inicio visual
  logo-instal.png   → logo/icono instalado o marca puntual
  splash.png        → lámina explicativa del flujo visual del splash
```

Reglas:
- no usar logos viejos;
- no cortar el logo;
- no deformar el logo;
- no mostrar formularios;
- no mostrar roles;
- no meter Home salvo como transición definida;
- no usar splash como screenshot runtime.
